---
title: "Chain Compatibility"
description: "Bluprynt is compatible with EVM (Ethereum Virtual Machine) and SVM (Solana Virtual Machine) blockchain networks."
---

Bluprynt is compatible with EVM (Ethereum Virtual Machine) and SVM (Solana Virtual Machine) blockchain networks. Currently, Smart Disclosures and KYI directly support the following networks:

| Network | Product Support |
|---|---|
| Stellar | <a href="../smart-disclosures">Smart Disclosures</a> |
| Solana | <a href="../smart-disclosures">Smart Disclosures</a> and <a href="../kyi-know-your-issuer">KYI (Know Your Issuer)</a> |
| Avalanche | <a href="../smart-disclosures">Smart Disclosures</a> |
| Aptos | <a href="../smart-disclosures">Smart Disclosures</a> |
| Plume | <a href="../smart-disclosures">Smart Disclosures</a> and <a href="../kyi-know-your-issuer">KYI (Know Your Issuer)</a> |
| Sepolia | <a href="../smart-disclosures">Smart Disclosures</a> and <a href="../kyi-know-your-issuer">KYI (Know Your Issuer)</a> |
| Ethereum | <a href="../smart-disclosures">Smart Disclosures</a> and <a href="../kyi-know-your-issuer">KYI (Know Your Issuer)</a> |
| Base | <a href="../smart-disclosures">Smart Disclosures</a> and <a href="../kyi-know-your-issuer">KYI (Know Your Issuer)</a> |
| XRPL | <a href="../smart-disclosures">Smart Disclosures</a> and <a href="../kyi-know-your-issuer">KYI (Know Your Issuer)</a> |
