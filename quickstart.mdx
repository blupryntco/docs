This guide takes you from zero to a live Bluprynt account with an asset under supervision and a working KYI lookup.

## Step 1 — Create your account

<AccordionGroup>
  <Accordion icon="user-plus" title="Sign up and verify your organization">
    1. Go to [app.bluprynt.com](https://app.bluprynt.com) and choose **Sign up**.
    2. Enter your corporate email and password.
    3. Confirm your email, then return to complete your organization profile.
    4. Complete KYB (Know Your Business) via Sumsub — this is required before you can file disclosures or publish attestations.

    <Tip>
      Using a MiCA-regulated workflow? Use the dedicated [MiCA signup](https://app.bluprynt.com/signup-mica) flow for a streamlined experience.
</Tip>
</Accordion>
  <Accordion icon="building" title="Invite your team">
    1. Go to **Settings → Members**.
    2. Click **Invite members**, enter email addresses, and pick a role.
    3. Invitees receive a token link; accepting links their user to your organization.
</Accordion>
</AccordionGroup>

## Step 2 — File your first disclosure

<AccordionGroup>
  <Accordion icon="file-plus" title="Create a Smart Disclosure">
    1. From the dashboard, click **New document**.
    2. Pick your jurisdiction, regulation, and asset type — Bluprynt filters available forms to only those relevant.
    3. Upload any source documents; we'll extract structured data automatically with AI-assisted autofill.
    4. Review, validate, and **Submit**.
</Accordion>
  <Accordion icon="rocket" title="Publish on-chain">
    1. Once submitted, click **Publish**.
    2. Pick the chain(s) — Ethereum, Solana, Stellar, Aptos, or XRPL.
    3. The artifact is uploaded to IPFS, then an attestation is written on-chain.
    4. Use **View onchain data** to confirm the transaction.
</Accordion>
</AccordionGroup>

## Step 3 — Run your first KYI lookup

<AccordionGroup>
  <Accordion icon="id-card" title="Via the Bluprynt app">
    1. Go to the **KYI** section.
    2. Enter a wallet address or asset CCID.
    3. View the issuer's verified identity, outstanding attestations, and credential history.
</Accordion>
  <Accordion icon="code" title="Via the API">
    ```bash
    curl https://api.bluprynt.com/v1/kyi/{ccid} \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
    See the [API reference](/api-reference/kyi) for the full schema.
</Accordion>
  <Accordion icon="window" title="Via the widget SDK">
    ```bash
    npm install @bluprynt/kyi-widget-sdk
    ```

    ```typescript
    import { BlupryntKyiWidget } from '@bluprynt/kyi-widget-sdk';

    <BlupryntKyiWidget accessToken={token} />
    ```
    Full integration guide: [KYI Widget](/integration-guides/kyi-widget).
</Accordion>
</AccordionGroup>
