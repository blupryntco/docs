Smart Disclosures is designed to integrate seamlessly with platforms that rely on verifiable disclosures for token listings, regulatory compliance, or DeFi protocol participation.

Attestations can be:

* Embedded in token metadata
* Queried via API for gating smart contract actions
* Displayed in explorer interfaces for compliance visibility
* Submitted to centralized exchanges as proof of disclosure readiness

***

### Ideal Integration Partners

We actively support integration with:

* **Centralized Exchanges (CEXs)**: to streamline token listing due diligence
* **Regulatory Sandboxes**: for automated document intake and approval tracking
* **DeFi Protocols & Launchpads**: to enforce minimum disclosure standards for participation
* **Block Explorers & Wallets**: to display disclosure status and prevent user interaction with non-compliant tokens
